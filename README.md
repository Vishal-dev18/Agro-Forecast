## 📌 Overview

This project predicts crop yield using Machine Learning and provides intelligent recommendations for farmers.

---

## 🚀 Features

* 🔐 Login & Authentication
* 🤖 ML-based Yield Prediction
* 🌱 Crop Recommendation System
* 📊 Visualization Dashboard
* 📥 Download Results
* 📜 Prediction History
* 👑 Admin Panel

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* SQLite

---

## 📂 Project Structure

```
data/        → Dataset  
src/         → ML code  
models/      → Saved models  
app/         → Streamlit app  
outputs/     → Graphs  
report/      → Documentation  
```

---

## ▶️ Run Project

```bash
pip install -r requirements.txt
python convert_data.py
python src/train_full_model.py
streamlit run app/app.py
```
